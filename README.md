# Snipper Pro

A professional Excel add-in for extracting and managing data from documents. Built with C# and WebView2.

## Features

- Document viewing with WebView2
- Text and table extraction from PDFs and images
- Excel integration via VSTO
- Modern UI with professional styling
- Table detection and extraction
- Multi-page document support

## Requirements

- Windows 10 or later
- Microsoft Excel 2016 or later
- WebView2 Runtime
- .NET Framework 4.8

## Installation

1. Run `install-excel-addin.bat` as administrator
2. The add-in will be installed and registered
3. Open Excel to use the DATASNIPPER tab

## Development

Built with:
- Visual Studio 2022
- WiX Toolset for MSI creation
- WebView2 SDK
- VSTO for Excel integration
